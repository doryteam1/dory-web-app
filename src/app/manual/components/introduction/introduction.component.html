1. ¿QUÉ ES EL SITIO WEB DORYWEBSITE?

El sitio web DORYWEBSITE es un software orientado a la consolidación de una comunidad de negocios electrónicos en el contexto de la actividad piscícola desarrollada en la Región Caribe Colombiana. Dicho sitio web brinda los servicios de registro y consulta de actores asociados al sector piscícola (piscicultores, pescadores, asociaciones, investigadores, transportadores, distribuidores y comercializadores), asesoría e interacción en tiempo real, oferta online de productos-servicios, y por último, geolocalización de granjas productoras y organizaciones piscícolas.



2. ¿POR QUÉ USAR EL SITIO WEB DORYWEBSITE?

El sitio web DORYWEBSITE busca, en primer lugar, dinamizar el intercambio de información entre los diferentes actores de la comunidad de piscicultores del Departamento de Sucre. El presente manual es hace parte del material didáctico diseñado para mejorar la experiencia de usuario y motivar a los interesados en la plataforma para que aporten ideas que permitan la sostenibilidad de este proyecto.



3. ¿CÓMO USAR ESTE MANUAL?

Este manual describe el proceso de navegación al interior del sitio web DORYWEBSITE mediante un enfoque gráfico centrado en imágenes y captura de pantallas. Adicionalmente, se presentan textos cortos que describen lugares, enlaces, así como acciones a que se deben realizar para ingresar y obtener información. Finalmente, se describen la forma para gestionar los diferentes servicios que implementa dicho sitio web.
